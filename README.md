# Creature Forge

Creature Forge is a fantasy-inspired creature generation application built using HTML, CSS, and JavaScript. Users can combine different creature traits, elements, and habitats to create unique mythical beasts complete with names, lore, rarity levels, power ratings, and classifications.

Designed as a magical creature laboratory, the application transforms simple trait combinations into fully generated fantasy creatures that can be collected and stored within a personal bestiary.

## Features

- Custom creature creation system
- Multiple head and body combinations
- Element selection system
- Habitat selection system
- Procedural creature name generation
- Dynamic lore generation
- Rarity system
- Power level calculation
- Creature class assignment
- Random creature generator
- Bestiary collection
- Favorite creature system
- Local Storage support
- Creature comparison panel
- Responsive design
- Fantasy-themed interface

## Generated Information

Every forged creature receives:

- Unique Name
- Rarity
- Power Level
- Creature Class
- Element
- Habitat
- Lore Description
- Power Tier Ranking

No two creations need to feel exactly the same.

## Rarity System

Creatures can belong to one of six rarity levels:

⚪ Common

🟢 Uncommon

🔵 Rare

🟣 Epic

🟡 Legendary

✨ Mythical

Higher rarities generally result in stronger and more valuable creatures.

## Power Rankings

Creatures are automatically ranked based on their power level:

- S Tier
- A Tier
- B Tier
- C Tier

## Bestiary

All generated creatures are stored in a personal Bestiary.

Features include:

- Creature history
- Favorite creatures
- Rarity tracking
- Power comparisons
- Collection management

Data is saved using Local Storage.

## Creature Comparison

The application automatically tracks:

- Strongest Creature
- Rarest Creature
- Average Power Level

Helping users compare their creations and build the ultimate collection.
